# Список сокращений

API — Application Programming Interface, программный интерфейс приложения

Argon2 — алгоритм хэширования паролей

AWS — Amazon Web Services

bcrypt — алгоритм хэширования паролей

BPMN — Business Process Model and Notation, нотация моделирования бизнес-процессов

CRM — Customer Relationship Management, управление взаимоотношениями с клиентами

CRUD — Create, Read, Update, Delete, создание, чтение, обновление, удаление

DFD — Data Flow Diagram, диаграмма потоков данных

DevOps — Development and Operations, практики разработки и эксплуатации

Docker — платформа контейнеризации приложений

DTO — Data Transfer Object, объект передачи данных

ER — Entity-Relationship, модель «сущность–связь»

ERP — Enterprise Resource Planning, планирование ресурсов предприятия

Express — веб-фреймворк для Node.js

FR — Functional Requirement, функциональное требование

FR-01 … FR-12 — коды функциональных требований

HRM — Human Resource Management, управление персоналом

HTTP — HyperText Transfer Protocol, протокол передачи гипертекста

IEC — International Electrotechnical Commission, Международная электротехническая комиссия

ISO — International Organization for Standardization, Международная организация по стандартизации

IT — Information Technology, информационные технологии

JSON — JavaScript Object Notation

JWT — JSON Web Token, веб-токен на основе JSON

MVP — Minimum Viable Product, минимально жизнеспособный продукт

multi-tenant — многоклиентский подход или архитектура

multi-tenancy — многоклиентская архитектура, одна платформа обслуживает несколько организаций-клиентов

NestJS — фреймворк Node.js для серверных приложений

Next.js — фреймворк React для веб-приложений

NFR — Non-Functional Requirement, нефункциональное требование

NFR-01 … NFR-09 — коды нефункциональных требований

NIST — National Institute of Standards and Technology, Национальный институт стандартов и технологий США

Node.js — серверная платформа JavaScript

ORM — Object-Relational Mapping, объектно-реляционное отображение

OWASP — Open Web Application Security Project

POST — HTTP-метод создания ресурса

PostgreSQL — реляционная система управления базами данных

Prisma — ORM для Node.js и TypeScript

RBAC — Role-Based Access Control, ролевая модель управления доступом

React — библиотека для построения пользовательских интерфейсов

REST — Representational State Transfer, архитектурный стиль взаимодействия через HTTP

SaaS — Software as a Service, программное обеспечение как услуга

shared database, shared schema — общая база данных и общая схема с логическим разделением по организации

SLA — Service Level Agreement, соглашение об уровне обслуживания

SQL — Structured Query Language, язык структурированных запросов

SSO — Single Sign-On, единый вход

SWOT — Strengths, Weaknesses, Opportunities, Threats, сильные и слабые стороны, возможности и угрозы

tenant — арендатор, клиент SaaS-платформы; в работе соответствует организации

tenant isolation — изоляция данных арендатора (организации)

tenant-контекст — контекст организации текущего пользователя (organization_id / organizationId)

TypeORM — ORM для TypeScript и JavaScript

TypeScript — типизированное надмножество JavaScript

UI — User Interface, пользовательский интерфейс

UML — Unified Modeling Language, унифицированный язык моделирования

UX — User Experience, пользовательский опыт

БД — база данных

ВКР — выпускная квалификационная работа

ИТ — информационные технологии

ПО — программное обеспечение

СУБД — система управления базами данных
